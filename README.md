## About The Project
Encode Base is a user-friendly desktop application that simplifies the process of encoding text.

## Getting Started

### Prerequisites

* Rust - 1.79 or above

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:nishidabruno/encode-gui.git
   ```
   
2. Build the project
   ```sh
   cargo build --release
   ```
